Primeira coisa que vamos fazer:

- Definir o Layout do site - Mock Flow (Estrutura com cabeçalho, barra de menus, conteúdo principal com artigo e uma sessão extra, rodapé)

- Definir a Paleta de Cores - Adobe Color (Paleta monocromática com cores em tons de marrom)

- Definir as Fontes - Google Fonts e Fonts Ninja (Roboto (arquivo .ttf) para Conteúdo do Site, Averia Sans Libre para o Título no cabeçalho e Bebas Neue para os demais Títulos)

-----------------------------------------------------------------------------------------------
Na prática:
1. Link para o Favicon e Chamar o arquivo CSS externo
2. Alimentar o site com conteúdo no HTML, utilizando Tags Semânticas
3. Organizar o conteúdo:
- Títulos e subtítulos
- Parágrafos
- Listas
- Imagens
- Vídeo Externo
- Links
- Emojis
- Formatações de textos
- Abreviações
4. Jogando Cores e Fontes por meio de Variáveis :root {--nome-var: }
5. Tornando a página responsiva ao redimensioná-la
6. Aprimorando o Cabeçalho <header> e o Menu de navegação <nav>
7. Aprimorando o Conteúdo Principal <main>
8. Aprimorando o Conteúdo Periférico <aside> e o Rodapé <footer>
9. Tornando o vídeo responsivo
